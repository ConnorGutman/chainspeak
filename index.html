What is this?

@chainspeak is a twitter account with no real owner. Anyone can tweet from it using a very tiny bitcoin transaction.

How is chainspeak anonymous?

Your message is kept in a bitcoin transaction which gets permanently recorded on the blockchain. Every few minutes @chainspeak will check for new transactions and tweet the latest messages!
